# Video Reconstruction

This is a research repository aimed at studiying the possibility of doing 3D reconstructions based on AV1's video codec motion vectors from inter prediction,luma predictions in intra prediction and other kinds of video encoding data that can be aplied to computer vision. An overview can be found [here](ideas.md). A proper paper is being written.

For the source code check [source code](src/), for the todo list check [todo](todo.md), for the information sources [sources](sources.md).
